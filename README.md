# Shabbat Blocker Script

סקריפט JavaScript פשוט ויעיל המאפשר להשבית את הגישה לאתר בשבתות ובמועדי ישראל על פי שעון ירושלים.

## מה הסקריפט עושה?
1. שולח שאילתה ל-API של [Hebcal](https://www.hebcal.com/) כדי לבדוק את הסטטוס ההלכתי הנוכחי בירושלים.
2. בודק את המשתנה `isAssurBemlacha`.
3. במידה והזמן מוגדר כ"אסור במלאכה" (שבת או חג):
   - מציג שכבת כיסוי (Overlay) לבנה ואטומה על כל המסך.
   - מונע אינטראקציה וגלילה באתר.
   - מציג הודעה: **"הדף אינו פעיל בשבת\ימים טובים לפי שעון ירושלים"**.
4. במידה ואין איסור מלאכה, הסקריפט לא מבצע שום פעולה והאתר פועל כרגיל.

## אופן השימוש
כדי להטמיע את הסקריפט, יש להוסיף את שורת הקוד הבאה בתוך תגית ה-`<head>` או בסוף תגית ה-`<body>` באתר שלכם.

## טיפול בשגיאות
במידה וישנה תקלה בתקשורת מול ה-API, הסקריפט לא יחסום את האתר כדי למנוע חסימות שווא, וישלח הודעה לקונסול: *"דף זה שומר שבת, אך לא הצלחנו לברר האם שבת עכשיו"*.
